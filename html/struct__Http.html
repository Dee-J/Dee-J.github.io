Detailed Description

Http 모듈에 대한 구조체이다. Http 모듈은 다양한 방식으로 Http 통신을 할 수 있다.

Note
Http의 Http 모듈에 대한 구조체이다.
구조체를 사용하기 전에 NewHttp() 함수를 사용해야 하며 사용이 끝났을 때 DestoryHttp() 함수를 꼭 사용해야 한다.
