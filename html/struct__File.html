Detailed Description

File 모듈에 대한 구조체이다. File 모듈은 다양한 방식으로 파일을 제어 할 수 있다.

Note
File의 File 모듈에 대한 구조체이다.
구조체를 사용하기 전에 NewFile() 함수를 사용해야 하며 사용이 끝났을 때 DestroyFile() 함수를 꼭 사용해야 한다.
