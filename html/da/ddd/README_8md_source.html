5 ## The Easiest Tizen Native Library: *Do It Tizen! (D.I.T.)*
6 ### Welcome to the *D.I.T.* GitHub page.
7 D.I.T. library is a light-weight library for manipulating Tizen Native API. Our goal is to make Tizen Native coding simpler, easier, and more fun!
8 
9 ## Why use this library?
10 1. We're **not used to** developing Tizen Native.
11 2. We need more easier Tizen Native Library.
12 3. ~~Some of sample codes of Tizen Native in 'tizen.org' is not working.~~
13 4. D.I.T. library makes Tizen seems like Android.
14 5. D.I.T. is heavily customizable.
15  * If you want to edit D.I.T. for your taste, **please follow D.I.T. Library Rules**
